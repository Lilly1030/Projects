# Dream Basket

> E-commerce site

## Usage


# Install dependencies
npm install

# Install client dependencies
cd client
npm install
cd ..
# Install admin dependencies
cd admin
npm install

# Run client and server (Visit http://localhost:3000)
npm run dev

# Run admin(:3001)
# first nagivate into admin folder
cd admin
npm start


# Run server only (:5000)
npm run server

# Run client only (:3000)
npm run client
